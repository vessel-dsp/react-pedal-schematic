# Vessel DSP Core Usage

`@vessel-dsp/core` converts between audio-circuit source documents and
tscircuit Circuit JSON. It does not ship a reusable React renderer, a custom
editor, or a realtime simulator.

## Parse And Export

```ts
import {
    parseCircuitDocumentFile,
    serializeCircuitJsonDocument,
    serializeCircuitDocumentFile,
} from '@vessel-dsp/core';

const document = parseCircuitDocumentFile(sourceText, {
    filename: 'boost.schx',
});

const circuitJson = serializeCircuitJsonDocument(document);

const asc = serializeCircuitDocumentFile(document, {
    format: 'ltspice-asc',
    filename: 'boost.asc',
});
```

## Convert Files

```ts
import { convertCircuitDocumentFile } from '@vessel-dsp/core';

const vdsp = convertCircuitDocumentFile(circuitJsonText, {
    inputFilename: 'boost.circuit.json',
    outputFormat: 'vdsp',
    outputFilename: 'boost.vdsp',
});
```

## Format Contract

V1 bidirectional Circuit JSON conversion covers:

- `.vdsp` strict `circuit-interchange/v2` YAML
- LTspice `.asc`
- LiveSPICE `.schx`
- tscircuit `.circuit.json`

SPICE `.cir` / `.net` support remains available for legacy connectivity parsing
and netlist export, but it is not part of the new v1 bidirectional Circuit JSON
promise.

## GitHub Pages

The GitHub Pages output is a static API reference generated by
`bun run build:pages`. It documents the core conversion functions only. Render
or edit emitted Circuit JSON in a downstream tscircuit application.
